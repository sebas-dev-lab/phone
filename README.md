## START PROJECT ##

- Instalar:

```npm install```

- Crear archivo .env y establecer las siguientes variables

REACT_APP_SIP_SERVER=
REACT_APP_AUTHORIZATION_USER=
REACT_APP_AUTHORIZATION_PASSWORD=
REACT_APP_DISPLAY_NAME=
REACT_APP_USER_AGENT=
REACT_APP_STUN_SERVERS=
REACT_APP_LOG_LEVEL=
REACT_APP_AVATAR=
REACT_APP_NUMBER=

- Compilar

```npm start```
